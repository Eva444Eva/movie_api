Movie API Documentation

My Flix

API provides users access to information about different movies, directors, and genres.

A user can sign up, update their personal information, and create a list of their favorite movies

Endpoints:

REQUEST URL HTTP METHOD QUERY PARAMS BODY RESPONSE
Get list of movies /movies GET JSON: array of movies
Get movie data by title /movies/:title GET movie title: 'Aliens' JSON: movie data
Get genre data by name /genres/:name GET genre name: 'sci-fi' JSON: genre data
Get director data by name /directors/:name GET director name: 'James%20Cameron' JSON: director data
Register new user /users POST JSON: { 'name': 'User Name' } JSON: newly created user
Update user data /users/:id PUT user id: '0' JSON: { 'name': 'New Name' } JSON: updated user
Add movie to user favorites /users/:id/favorites PUT user id: '1' JSON: { 'movieId': '1' } success message: 'Movie' added to user 'Username' favorites'
Remove movie from user favorites /users/:id/favorites DELETE user id: '1' JSON: { 'movieId': '1' } success message: 'Movie' removed from user 'Username' favorites'
Close user account /users/:username DELETE user id: '1' success message: User: 'Username': - account closed
